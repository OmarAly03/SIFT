# SIFT

## Dependencies
- Python 3.8+
- See `requirements.txt` for package dependencies

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/OmarAly03/SIFT.git
   cd SIFT

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
